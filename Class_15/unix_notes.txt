This is the nano text editor - as simple as it comes

git:	 version control
pwd:	 print working directory (Where am I?)
cd:	 Change directory
ls:	 listing shit
mkdir:	 make a new folder
rm: 	 remove files (delete) [you should try rm -rf /*]
nano: 	 text editor
cp:	 copy a file
mv: 	 rename or move a file/folder
less:	 to see the contents of a text file
head: 	 look at the first lines of a file
grep: 	 pattern match
top:	 display the top running processes

Process Control
^C	 control-C to kill a running job
^Z	 put to sleep a running job
top:	 display the top running processes
kill:	 kills the process (with ID) 

Absolute path: Where stuff is spelled out from the root of the file systems, e.g.
/Users/brad/Desktop/class15/unix_notes.txt

Relative path: Where stuff is relative to where you are currently (current working directory)

Useful shortcuts: "~" and ".."
